### **心得**

Lab02應該會是寫最多版的一次Lab

因為如果寫完一版，又想到一個可以跑更快的演算法，就又會需要重打一份 (所以打code之前真得先好好想演算法)

但也不推薦一版都沒有就一直抱頭想演算法 (如果想一個難的演算法，結果打不出來，裂開........)
